Keep track the users login. The plugin allows to save the data of each user's login (IP, username, date, time). 

== Installation ==
1. Upload/install the plugin.
2. Activate plugin.
3. Enjoy!

== Description ==
This plugin creates a Dashboard Widget and a admin page to display IP, username, date and time of each user login.
Only administrators can see this information on admin panel.
